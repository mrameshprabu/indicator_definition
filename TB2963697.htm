Background

Value Added Production Index (VAPI) measures changes in volume of output at close and regular intervals over a given reference period.

VAPI is compiled by excluding intermediate goods. This means that VAPI will reflect the actual value of production without intermediate goods.

The theoretical formula for the index is a Laspeyres method. VAPI is utilized to evaluate the country�s macro-economy. The index compiled by the Office of Industrial Economics is relative to the index conducted by the National Economic and Social Development Board.

 Release Schedule

The value added production value index is compiled and released at monthly interval. Preliminary monthly value added production value index are released a month after the end of the reference month. The revised figures are released a month later together with the preliminary figures for the next reference month. The statistics are announced through press release to the public and are also posted on the Office of Industrial Economics website on the same day.
